# 档案管理系统简介 java419

## 项目概述

本系统是一款基于Spring Boot与Vue.js的档案管理系统，旨在提供企业内部设备、员工、客户信息的高效管理。系统集成了设备管理、维修保养、配件采购等多个功能模块，助力企业提升管理效率。

## 技术栈

- **后端框架：** Spring Boot + MyBatis
- **前端框架：** Vue.js
- **数据库：** MySQL
- **项目构建工具：** Maven

## 功能模块

- **系统首页：** 快速导航至各管理模块。
- **个人中心：** 管理个人资料与账户设置。
- **员工管理：** 对员工信息进行维护。
- **客户信息管理：** 管理客户资料。
- **设备类型管理：** 设备类型与规格的管理。
- **设备型号管理：** 设备型号的详细管理。
- **设备信息管理：** 设备的详细信息管理。
- **设备维修管理：** 设备维修进度与记录管理。
- **设备保养管理：** 设备保养计划与执行管理。
- **定检信息管理：** 定期检查信息的管理。
- **配件信息管理：** 设备配件信息的维护。
- **配件采购管理：** 配件采购流程的管理。
- **合同信息管理：** 合同信息的归档与管理。

## 系统角色

- 系统管理员：负责系统的维护与权限分配。
- 员工用户：根据权限使用系统功能。

## 运行环境

- **后端：**
  - JDK 1.8 或以上版本
  - MySQL 5.7 或以上版本
- **前端：**
  - 现代浏览器

## 部署步骤

1. 准备后端运行环境，安装JDK与MySQL。
2. 使用Maven构建项目，生成可执行jar包。
3. 部署前端项目至Web服务器。
4. 配置数据库连接。
5. 启动后端服务。

## 目录结构

```
├── backend                后端代码目录
│   ├── src                源代码目录
│   ├── pom.xml            Maven项目配置文件
└── frontend               前端代码目录
    ├── src                源代码目录
    ├── public             静态资源目录
    ├── vue.config.js      Vue.js配置文件
```

## 核心亮点

- **前后端分离：** 前后端分离的设计，便于项目维护与扩展。
- **功能模块化：** 各功能模块独立，便于定制化开发。
